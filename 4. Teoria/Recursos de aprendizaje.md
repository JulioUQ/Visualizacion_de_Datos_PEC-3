# Análisis visual

En este bloque se introduce el uso de técnicas de visualización de datos como mecanismo para el análisis exploratorio de un conjunto de datos proporcionado.

Primero se recomienda leer un artículo que describe los conceptos básicos relacionados con la analítica visual, muestra algunos trabajos históricos como el cuarteto de Anscombe o el mapa del cólera de John Snow, y diferentes categorías de herramientas que pueden ser usadas para extraer conocimiento de los datos de forma visual, como primer paso para contar una historia. Los humanos tenemos un sistema visual que permite identificar patrones, tendencias, anomalías, etc. de forma casi instantánea, así que el uso de visualizaciones de datos puede ser muy útil para preparar y entender los datos.

Después, un vídeo desarrolla el concepto de analítica visual, presentándolo como un proceso iterativo en el que los datos son inspeccionados mediante visualizaciones de datos, revelando aspectos interesantes que pueden formar parte de la historia a narrar. Para ello se revisan los ejemplos mencionados en el artículo anterior, haciendo hincapié en que la analítica visual no es una herramienta, sino un proceso que obliga a ir adelante y atrás, replanteando los datos, los objetivos deseados y el procedimiento para obtenerlos.

A continuación, otro vídeo introduce el concepto de interactividad en la visualización de datos. El uso de visualizaciones de datos interactivas es un paso adelante al integrar en la propia visualización la capacidad de explorar los datos, hacerse las preguntas pertinentes, y narrar una historia de forma guiada. El vídeo describe las operaciones más comunes que se pueden realizar con una visualización de datos interactiva, como ejemplo de lo que se puede llegar a realizar.

Por último, un material docente muestra cómo usar una herramienta como D3 para analizar datos de forma interactiva mediante una visualización de datos, aprovechando las posibilidades que ofrece la herramienta. Aunque D3 ha evolucionado en lo que ahora se conoce como Observable, este material docente sigue siendo válido para plantearse el procedimiento a seguir para elegir una técnica gráfica en función de la naturaleza de los datos, con una recopilación de las posibilidades de D3, especialmente en lo que se refiere a su interactividad y dinamismo. Este recurso puede serviros como inspiración para pensar en todo lo que se puede hacer mediante una visualización de datos interactiva.

# Storytelling

Hans Rosling destacó por su destreza en el arte del storytelling aplicado a la visualización de datos. Su enfoque innovador integró técnicas avanzadas de visualización con narrativas cautivadoras, transformando conjuntos de datos complejos en relatos persuasivos y accesibles. Rosling defendió el storytelling como herramienta esencial en la democratización del conocimiento estadístico, evidenciando cómo la combinación de datos y narrativa puede catalizar una comprensión más profunda y significativa de los problemas globales.

En uno de sus famosos vídeos, Rosling comenta: "Tener los datos no es suficiente, hay que mostrarlos de forma que la gente los disfrute y entienda". En este bloque se trabaja el vídeo 200 países, 200 años, 4 minutos de Hans Rosling y la lectura de los artículos relacionados. El primero de ellos hace una revisión detallada de lo que nos va presentando el vídeo, mientras que los otros dos presentan diferentes aspectos relacionados con el enfoque que toma Rosling (¡siempre positivo!) y la estructura adecuada para hacerlo.

Después hay una lectura recomendada en la que se describen las fases del proceso de diseño a seguir para crear una visualización de datos. Aunque no seáis diseñadores, este recurso os puede ser útil para ordenar vuestras ideas. Finalmente, una última lectura hace énfasis en que para narrar una historia no es suficiente con mostrar datos, sino que debe haber una clara intención detrás, ligada al objetivo de la visualización de datos que se quiere crear.
